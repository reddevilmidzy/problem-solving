### 문제 설명

<p>쿠는 <span class="wave-text">KUPC 2025</span>에 참가하는 학생을 대회장으로 안내하는 역할을 맡았다. 건국대학교는 무한히 큰 $2$차원 격자로 나타낼 수 있고, $(X, Y)$에서 대회가 진행된다.</p>

<p>올해는 $N$명의 학생이 <span class="wave-text">KUPC 2025</span>에 참가하기 위해 대회장으로 향하고 있다. $i$번째 학생의 초기 위치는 $(x_i, y_i)$이고, <strong>모든 학생은 $x$축 또는 $y$축과 평행하게만</strong> 움직일 수 있다!</p>

<h5 style="text-align: center;"><img alt="" height="394" src="" width="397"></h5>

<h5 style="text-align: center;">두 학생이 대회장으로 이동하는 예시</h5>

<p>쿠는 $N$명의 학생에게 각각 몇 번 이동 방향을 바꾸어야 대회장에 도착할 수 있는지 알려주려고 한다. 쿠를 도와 학생을 안내해 보자! 단, 학생이 초기 위치에서 처음 움직이는 것은 이동 방향을 바꾼다고 생각하지 않는다.</p>

### 입력 

 <p>첫째 줄에 대회장의 위치를 나타내는 정수 쌍 $(X, Y)$가 공백으로 구분되어 주어진다. $(-10^9 \le X, Y \le 10^9)$</p>

<p>둘째 줄에 <span class="wave-text">KUPC 2025</span>에 참가하는 학생의 수를 나타내는 정수 $N$이 주어진다. $(1 \le N \le 100)$</p>

<p>셋째 줄부터 $N$개의 줄에 걸쳐, $i+2$번째 줄에 $i$번째 학생의 초기 위치를 나타내는 정수 쌍 $(x_i, y_i)$가 공백으로 구분되어 주어진다. $(-10^9 \le x_i, y_i \le 10^9)$</p>

<p>대회장과 모든 학생은 각각 서로 다른 위치에 존재한다.</p>

### 출력 

 <p>$N$개의 줄에 걸쳐, $i$번째 줄에 $i$번째 학생이 대회장에 도착하기 위해 이동 방향을 바꿔야 하는 최소 횟수를 출력한다.</p>
